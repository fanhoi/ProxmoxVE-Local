# История изменений (CHANGELOG)

Все заметные изменения в этом проекте будут задокументированы в данном файле.

## [Невыпущенные изменения]

### Изменения в документации
- Переведен файл `README.md` на русский язык для удобства русскоязычных пользователей.
- Удалена плашка о поиске мейнтейнеров в `README.md`.
- Ссылки на репозиторий проекта в `README.md` обновлены на форки пользователя (`fanhoi/ProxmoxVE-Local` и `fanhoi/ProxmoxVE`), в том числе ссылка на скрипт автоматической установки.

### Скрипты установки (Вариант 3)
- Созданы новые промежуточные скрипты для установки веб-панели из форка:
  - `pve-scripts-local.sh` — скрипт инициализации установки на хосте Proxmox (создает контейнер и перехватывает запросы `curl` к установщику).
  - `pve-scripts-local-install.sh` — установщик для выполнения внутри контейнера, который устанавливает `git` и клонирует репозиторий `fanhoi/ProxmoxVE-Local.git` напрямую.

### Изменения в коде
- Выполнена глобальная замена ссылок с оригинального репозитория `community-scripts` на форки пользователя `fanhoi` в следующих компонентах и сервисах:
  - `Footer.tsx` (футер приложения)
  - `HelpModal.tsx` (модальное окно помощи)
  - `version.ts` (проверка обновлений и обновление скрипта `update.sh`)
  - `repositoryService` (управление репозиториями)
  - `githubJsonService` (синхронизация скриптов)
  - `scriptDownloader.js` (загрузка скриптов с GitHub)
  - `pre-release-updater.sh` (скрипт обновления предварительных релизов)
- Частично переведен интерфейс главной страницы (`page.tsx`): локализованы названия вкладок меню ("Доступные скрипты", "Загруженные скрипты", "Установленные скрипты", "Резервные копии", "Генератор") и шапка сайта ("Management" -> "Локально").
- В левый верхний угол панели добавлена метка `(Форк fanhoi)` для визуального контроля установки.
